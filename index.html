Important Notes What is HTML?

HTML stands for Hypertext Markup Language, and is the main document that the internet is made up of. It's what makes it able for you to see what a webpage says (text) and how it looks (markup) . CSS makes it look a tad more pretty and is basically the stylist of HTML is how I like to think of it. Here is a more in depth explanation of HTML


What Are Tags And Elements?

Tags and Elements are what we tell the HTML document to do. For instance, In order to create the "Important Notes" text at the top and have it formatted like it is, we used the h1 tag which stands for Heading #1 (there are multiple styles of headings in HTML). Here is a more in depth explanation of tags and elements

So what's the difference between Elements and Tags? Well Elements are defined by a starting tag (element) and if it has additional content then it is closed with an ending tag (/element) . There are things called void elements however that do not need a closing tag and are just ended with a /> .



So How Does The Internet Work?

The internet is a bunch computers that communicate with each other each time someone tries to connect to a website. When they do try the user's computer sends a HTTP request to a server to attempt to find what the user wants and the server sends back the appropriate HTML document that the user's browser can portray for us humans to see.
